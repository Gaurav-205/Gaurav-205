- Real-time chat with around 100 ms latency
- Google Maps-based campus navigation
- CI/CD deployment pipeline reduced release time to under 3 minutes

### Sahara - Pet Care App

- Production-ready mobile app for pet booking workflows and service management
- Real-time chat and media upload support
- End-to-end flow for search, schedule, and booking confirmation
- Store-ready architecture with secure auth and notifications
